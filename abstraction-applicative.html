Applicative

A functor with application, providing operations to embed pure expressions (return), and sequence computations and combine their results (<*>). _

Minimal complete definition

  • return x/result x
  • (<*>) f x
1: 
2: 

static member Return (x:'T) : 'Applicative<'T>
static member (<*>) (f: Applicative<'T->'U>, x: Applicative<'T>) : Applicative<'U>

Note: return can't be used outside computation expressions, use result instead.

Rules

1: 
2: 
3: 
4: 

result id <*> v = v
result (<<) <*> u <*> v <*> w = u <*> (v <*> w)
result f <*> result x = result (f x)
u <*> result y = result ((|>) y) <*> u
