Validation<'err, 'a>

Namespace: FSharpPlus.Data

A 'Validation' is either a value of the type 'err or 'a, similar to 'Result'. However, the 'Applicative' instance for 'Validation' /accumulates/ errors using a 'Semigroup' on 'err. In contrast, the Applicative for 'Result' returns only the first error.

A consequence of this is that 'Validation' is not a monad. There is no F#+ 'Bind' method since that would violate monad rules.

An example of typical usage can be found here.
