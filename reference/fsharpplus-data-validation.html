bind f x
Signature: f:('T -> Validation<'Error,'U>) -> x:Validation<'Error,'T> -> Validation<'Error,'U>
Type parameters: 'T, 'Error, 'U

Binds through a Validation, which is useful for composing Validations sequentially. Note that despite having a bind function of the correct type, Validation is not a monad. The reason is, this bind does not accumulate errors, so it does not agree with the Applicative instance.

There is nothing wrong with using this function, it just does not make a valid Monad instance.
